# 🗂️ Task Manager App

A modern and responsive Task Management application built with **Next.js, TypeScript, and Redux Toolkit**.  
This project demonstrates real-world frontend development skills, scalable architecture, and state management.

---

## 🚀 Features

- Create, edit, and delete tasks
- Organize tasks by status
- Fully responsive UI
- Scalable component architecture (Atomic Design)
- State management with Redux Toolkit
- Clean and modular code structure

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- React
- Redux Toolkit
- CSS / SCSS (or Tailwind if used)
- Git & GitHub

---

## 🧱 Architecture

This project follows **Atomic Design methodology**:

- atoms
- molecules
- organisms
- templates
